DokiMod 为开发者准备了一些官方插件，您可以借助这些插件来让自己的 Mod 更出色。

# 社区角色神态包

追加更多角色神态，并让 Yuri y / Insane / Yandere 系列神态支持日常服装。

# Blinky Flappers!

移植 实验性

向角色追加更多动画。（移植自 Blinky Flappers! Mod）

# 不要晴天娃娃！！！！！

实验性 bug 多

去除原游戏中的“晴天娃娃”，充分考虑 Sayori 厨的感受。当然，同时去除了 Yuri 拿刀自裁的东西。

该插件目前有已知问题。
